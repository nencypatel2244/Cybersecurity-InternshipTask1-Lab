# Cybersecurity Internship - Task 1: Lab Setup & Fundamentals

This repository documents the completion of Task 1 for my cybersecurity internship at ApexPlanet. 

## What was Accomplished:
1. **Virtual Lab Setup:** Configured an isolated virtual network environment using Oracle VirtualBox with Kali Linux (Attacker Machine) and Metasploitable 2 (Vulnerable Target Machine) connected via a secure Host-Only adapter.
2. **Network Verification:** Confirmed secure internal connectivity between virtual machines using network commands and analyzed the traffic live via Wireshark.
3. **Linux Fundamentals:** Practiced advanced navigation, file permissions modification (`chmod`, `chown`), and network troubleshooting configurations.
4. **Cryptography Basics:** Explored cryptographic hashing functions (MD5, SHA256) and successfully executed symmetric encryption/decryption using OpenSSL.

## Repository Contents:
- `linux_cheatsheet.md`: A curated cheat-sheet of essential Linux administration and cryptographic terminal commands.
- `Lab_Setup_Report.docx`: Detailed step-by-step report with screenshots validating the network configuration and tool verification.
